== Description ==

A simple slider that can easily be set up and used. Uses jQuery [Nivo Slider](http://nivo.dev7studios.com/ "The world's most awesome jQuery & WordPress Image Slider") script for slide animation and custom posts and custom taxonomies for slides management. The plugin can also use posts, pages, or other custom posts as slides. Supports shortcodes with a number of options to allow embeding in posts, pages, or theme files. **Please don't hesitate to contact the plugin author for questions, suggestions, etc.**

[Plugin Documentation](http://apocalypseboy.com/simple-slides/ "Simple Slides")


== Installation ==
